\section{模板使用指南}
\subsection{编写目的}
    当学者们使用Office书写数学论文时, 一个巨大的麻烦是数学公式的\red{输入与排版}. 一个是数学公式输入需要使用专门的软件(可以使用word自带的公式编辑器, 但强烈建议使用mathtype软件), 每个公式是作为图片插入到文档中, 所以查找替换公式里面的内容已成为一种不可能的事情. 另外一个是排版, 插入数学公式后, 文档的行间距就会产生变化, 无法做到全文一致, 失去美观性. 为此, 众多学术期刊杂志都使用Tex作为排版软件.

    为克服Office软件排版的缺陷, 本院采用Tex作为毕业论文的排版软件.
    为了简化排版, 也为了消除排版的随意性, 我们编写了此个模板共大家使用, 它符合学校对论文格式的要求.

\subsection{功能}
    本模板提供以下便利功能:
    \begin{itemize}
      \item 根据系统时间, 自动确定论文写作年份;
      \item 根据题目的长度, 自动排成居中的两行或者一行;
      \item 正文前的页面不要额外排版, 只需要提供相应信息即可;
      \item 公式、定理、标题, 全文自动编号, 无需手动编号;
      \item 提供simplify.def文档, 内含大量已定义好的宏命令, 便于简化输入.
    \end{itemize}

\subsection{模板使用}
    打开main.tex或者sim$\_$main.tex, 使用\red{pdflatex.exe}编译\red{两次}便产生最终文档. mian.tex提供了一篇完整的毕业论文, 而sim$\_$main.tex只是提供一个论文框架.

\subsection{文档构成}
    本论文模板由一个主文件\red{main.tex}(以及简化版sim\_main.tex和范例版exam\_main.tex)和三个文件夹\red{def}, \red{doc}和\red{photo}组成. 文件夹def包含以下文件
    \begin{multicols}{2}
    \begin{itemize}
      \item[1)] format.sty: 格式包, 引用\blue{package.def}, \blue{page.def}和\blue{config.tex}3个文档
      \item[2)] package.def: 调用的各种宏包
      \item[3)] page.def: 页面设置文档
      \item[4)] mytikz.def: 绘图命令, 含坐标绘制
      \item[5)] simplify.def: 简化命令文
      \item[6)] firstpage.tex: 封面
      \item[7)] copyright.tex: 版权页
      \item[8)] abstract.tex: 生成中英文摘要等
      \item[9)] school.jpg: 校名图片
    \end{itemize}
    \end{multicols}\noindent
    上述文档中, simplify.def放置了各种简化宏命令, 有助于用户简化输入. mytikz.def是绘图命令集合, 简化了坐标绘制.

    文件夹doc用来放置用户各章节文档, 本模拟文档包含了introduction(引言), section2(第二节), section3(第三节), section4, section5, references(参考文献), acknowledge(致谢), aprodix(附录).

    文件夹photo用来放置毕业论文所需要引用的图形文件. 所以本模板中, 文件以及文件夹的结构如下图\ref{fig1}所示.

    目录是由主文件的命令$\backslash$tableofcontents所生成, 按规定只显示二级标题.
    \subsection{段落格式}
    行文中, 段首是否顶格、缩进, 往往是初学者迷茫的地方. 如果行文逻辑上没有换段, 只是换行, 那么段首不缩进, 此时在段首加命令
    $\backslash$noindent. 如下面排版中, ``那么"这一句就不用缩进. \\[10pt]
    \frame{\parbox[b][2.35cm][c]{\textwidth}{\qquad我们已经证明了
	   $$a^2+b^2=c^2,$$
    那么就可以证明$c^2=a^2+b^2$."}}

    最忌讳的是, 一段还没有完成就换段. 如下排版是不适合的. ``那么"应该紧随前行\\[10pt]
    \frame{\parbox[b][2.35cm][c]{\textwidth}{\qquad我们已经证明了
	   $$a^2+b^2=c^2,$$
    \qquad那么就可以证明$c^2=a^2+b^2$."}}

    以下排版更是犯了大忌\\[10pt]
    \frame{\parbox[b][2.1cm][c]{\textwidth}{
    \qquad根据条件(1),

    \qquad和条件(2),

    \qquad以及定理2.1的结论, 我们便得到了结论.}}

    如果行文中确实是逻辑换段, 那么该段落首行应该缩进, 此时的操作是: \red{空一行}, 即输入两个\red{回车}符号.

    \subsection{字体}
    行文中, 中文使用宋体, 小四; 西文使用罗马字体, 小四（12磅）, 本模板已经自动设置了.

    在数学环境中, 会出现斜体和正体以及花体、空心字体等. 一般的原则是
    \begin{itemize}
      \item \red{正体}: 常数、函数名一律用正体, 如$1$, $2$, $\sin $, $\cot$, $\ln$, $\log$等;
      \item \red{斜体}: 变量一律使用斜体, 如$x$, $y$, $A$;
      \item \red{空心字体}: 一些特殊的集合使用空心字体, 如: 自然数$\N$, 整数$\Z$, 有理数$\Q$、实数$\R$, 复数$\mathbb{C}$.
    \end{itemize}

    圆周率$\pi$和自然对数底数e都是常数, 行文中应该使用正体, 但是由于犯错的人员太多了, 所以大家也认可它们的斜体形式.

    \subsection{编号}
    全文编号是自动进行的, 包括标题、公式、定理(引理、例等等)、参考文献。采用自动编号模式, 可以省去手工编号的麻烦, 更省去修改文章而导致重新编号的麻烦.

    \subsubsection{标题编号}
    标题编号, 分为自动编号和手工编号两种. 建议使用自动编号. 一般编号到四级标题. 三级(含)以上标题不出现在目录页.
    \subsubsection{公式编号}
    在文中需要引用的公式, 一定要进行标号. 本模板的公式编号是以节为前缀的. 进行公式标号的数学环境主要有: 单行公式, \red{equation}, 多行公式\red{eqnarray}和\red{gather}. 具体操作为
    \begin{spacing}{1.2}\zihao{-4}
    \begin{verbatim}
    若我们证明了
    \begin{equation}\label{eqabcd}
      a^2+b^2=c^2,
    \end{equation}
    那么我们就知道勾股定理成立.
    \end{verbatim}
    \end{spacing}\vspace*{-20pt}\noindent
    显示效果为\\[10pt]
    \framebox{\parbox{\textwidth}{\qquad 若我们证明了
    \begin{equation}\label{eqabcd}
      a^2+b^2=c^2+d^2,
    \end{equation}
    那么我们就知道勾股定理成立.}}\medskip

    上文中$\backslash$label$\{$eqabcd$\}$就是给公式\ref{eqabcd}一个名称, 便于后文引用. 这里的\ref{eqabcd}就是一个引用, 它实现的命令是$\backslash$ref$\{$eqabcd$\}$.

\subsection{标点符号}
    从摘要到参考文献, 文中的符号, 都应该用西文的半角符号. 建议双引号使用中文的全角符号. 如果在书写的时候, 使用的是中文标点, 可以替换. 注意替换范围（不包括声明页和后记页）. 西文符号后要跟一个空格. 行文中, 应该加上标点, 包括数学公式. 如：`` .",``;",``,"等等.

    数学中的省略号是三个点: $\ldots$和$\cdots$. 使用它们的规则是, 三个点的重心位置与前后的符号的重心一致. 如以下省略号使用是规范的,
    \cent[4]{
        \$\$1+2+$\backslash$cdots+n. $\backslash$quad i=1,2,$\backslash$ldots,n.\$\$}
    \noindent显示效果为\medskip

    \noindent
    \frame{\parbox[c]{\textwidth}{$$1+2+\cdots+n.\quad i=1,2,\ldots,n.$$}}

    中文的省略号是6个点$\cdots\cdots$, 可以使用中文输入法输入.

\subsection{引用}
    交叉引用的简洁性是Tex受欢迎的一个重要原因. 虽然Office也可以建立交叉引用, 但是它实现起来的操作较为繁琐, 很多人望之却步, 敬而远之. Tex的交叉引用就较为简单, 只需两步即可: \red{命名}和\red{引用}. 首先在需要引用的地方设置一个标识\red{label}, 然后在需要引用的地方输入它的名称.

    虽然引用初次使用很麻烦, 但是文章修改起来却是相当方便的. \red{不用不知道, 使用真巧妙}. 具体操作如下. 定理、公式、图表和章节的引用方式类似, 参考文献的引用方式特殊性些.
    \subsubsection{定义标签}
    在需要引用的地方, 输入$\backslash$label$\{$标签名称$\}$. 如下就将定理\ref{theo2}命名为\red{theo2}.\\[-20pt]
    \begin{spacing}{1.35}\zihao{-4}
    \begin{verbatim}
    \begin{theo}\label{theo2}
        若实数$a<3$, 那么$a<4$.
    \end{theo}
    \end{verbatim}
    \end{spacing}\noindent
    \frame{\parbox[b][1cm][c]{\textwidth}{
    \begin{theo}\label{theo2}
        若实数$a<3$, 那么$a<4$.
    \end{theo}}}

    公式的命名类似, 需要注意的是, 此时调用的数学环境是不带*的, 如 \red{equation}, \red{eqnarray}, \red{gather}等能自动编号的数学环境. 下面就给公式\ref{eqabc}命名为eqabc.\\[-20pt]

    \begin{verbatim}
    \begin{equation}\label{eqabc}
        \Big(\dfrac{a+b+c}3\Big)^3\ge abc.
    \end{equation}
    \end{verbatim}

    \noindent
\frame{\parbox[b][1.1cm][c]{\textwidth}{
    \begin{equation}\label{eqabc}
        \Big(\dfrac{a+b+c}3\Big)^3\ge abc.
    \end{equation}}}

    图表的命名需要调用\red{caption}命令, 详见上文\ref{subsub1}节.

    以下代码, 将该条\red{参考文献}命名为zhs.
    \begin{verbatim}
    \bibitem{zhs} 张三, 李四. 论学习[J]. 生活实践, (2020)\textbf{5}(33): 1-10.
    \end{verbatim}

    \subsubsection{建立引用}
    在需要引用定理、图表的地方, 直接输入\red{$\backslash$ref$\{$标签名称$\}$}; 对于公式引用, 输入\red{$\backslash$eqref$\{$标签名称$\}$}; 对于参考文献, 输入\red{$\backslash$cite$\{$参考文献标签名$\}$}. 如下代码就是对公式、定理和参考文献的引用.     \begin{verbatim}
    根据公式\eqref{eqabc}, 以及文献\cite{zhs}和定理\ref{theo2}, 我们便知道命题结论成立.
    \end{verbatim}

    \subsection{参考文献}
    理工科专业毕业论文（设计）不采用脚注形式进行注释, 需要进行注释的, 置于参考文献中, 但正文中应标注与参考文献对应的编号.

    参考文献要求有近年来的\red{外文文献}, 其中理科学生必须达到\textbf{\red{两篇}}, 否则不能参加校级和省级优秀学士学位论文的评选.

    参考文献应该标识出类型, 主要有以下几种.\vspace*{-10pt}
    \begin{multicols}{2}
    \begin{itemize}
      \item [J]: 期刊论文;
      \item [M]: 书籍;
      \item [C]: 论文析出集;
      \item [D]: 学位论文.
    \end{itemize}
    \end{multicols}
    以下是参考文献的源代码.\vspace*{-10pt}
    \begin{spacing}{1.3}\zihao{-4}
    \begin{verbatim}
